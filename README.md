# MavenTest
The maven version of the "JTesting" done weeks before, you can find it in my repository.



### Test


Clone:
```sh
git clone https://github.com/Rebecasarai/MavenTest.git
cd MavenTest

```

Compile:

```sh
mvn compile
```

And Test it:

```sh
mvn test
```
